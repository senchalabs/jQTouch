This demo is a full-screen app. Please add this app to your home screen and relaunch.

Below is a horizontally scrolling tabbar. The indexLists are on the “Big List” and “Little List” tabs. The indices are generated at runtime for any page with an “index” class using initListIndices();. Individual pages with the “index” class can be indexed with initListIndex('bigList');. The jqt.tabbar extension is required.


Touch Big List’s first entry to go to that entry’s detail page.


Tabs Blank 1, 2 & 3 are animated.


Built using jQTouch and iScroll v4.
